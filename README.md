## 🔍 3. Challenges & Resolutions

As an **AI-Native** specialist, encountering technical obstacles is an opportunity to showcase proactive problem-solving. Below are the specific issues faced during setup and how they were resolved:

### ⚠️ Challenge A: Resolving Extraneous Extension Matches in the Marketplace
*   **Issue:** Searching for "Claude Code" and "Codex" in the VS Code/Cursor Extension marketplace returned multiple matching results from different publishers, causing potential ambiguity about which was the official/correct plugin.
*   **Proactive Resolution:** 
    1. Carefully inspected the publisher names, ratings, download metrics, and verified badge statuses.
    2. Cross-referenced the extension descriptions with official documentations (Anthropic and Codex) to ensure the official wrappers were selected.
    3. Successfully installed the correct, verified extensions and completed the OAuth login flows securely.

### ⚠️ Challenge B: Windows Git Credential Manager Handshake Delay
*   **Issue:** During the initial `git push` command, Windows Git Credential Manager had a temporary protocol handshake timeout, causing the terminal command to hang.
*   **Proactive Resolution:** 
    1. Terminated the stalled process.
    2. Ran `git config --global credential.helper manager` to verify configuration.
    3. Used a Personal Access Token (PAT) with appropriate scopes to bypass credential popups and ensure immediate, secure authorization. Subsequent pushes succeeded instantly.

### ⚠️ Challenge C: Workspace Trust Limitations in Cursor
*   **Issue:** Upon opening the new folder, Cursor prompted for Workspace Trust, temporarily restricting full extension execution (like Claude Code) until trust was established.
*   **Proactive Resolution:** 
    1. Immediately marked the local directory as a "Trusted Workspace" to grant the AI integrations standard filesystem read/write privileges.
    2. Verified that all extension functionalities were fully active.
